# ArtBot - Ваш онлайн-ассистент по продаже картин

Добро пожаловать в **ArtBot**! Это ваш персональный бот для продажи уникальных произведений искусства. ArtBot помогает художникам и коллекционерам выставлять свои картины на продажу, а покупателям — находить и приобретать шедевры, которые украсят их пространство.

## ?? Функциональные возможности

- **Настройка бота:** Настройте бота под свои нужды. Этот бот изначально предназначен для продажи картин, но вы можете адаптировать его для продажи чего угодно.
- **Поиск картин:** Просматривайте и находите картины по различным критериям, таким как категория и подкатегория.
- **Продажа картин:** Легко добавляйте свои произведения искусства в базу данных и управляйте своими объявлениями.
- **Чат с ботом:** Получайте помощь и советы в режиме реального времени через удобный интерфейс чата.
- **Уведомления:** Получайте уведомления о картинах, которые выбрали пользователи.

## ?? Как начать

### Установка и настройка

1. **Склонируйте репозиторий:**

    ```bash
    git clone https://github.com/ваш-username/ArtBot.git
    cd ArtBot
    ```

2. **Установите зависимости:**

    ```bash
    npm install
    ```

3. **Создайте файл `.env` и добавьте необходимые переменные окружения:**

    ```env
    BOT_TOKEN=ваш_токен_бота
    ADMIN_IDS=id администраторов
    ```

4. **Запустите бота:**

    ```bash
    npm start
    ```

### Использование

После запуска бота вы можете взаимодействовать с ним через Telegram. Просто найдите бота по имени и начните чат.

## ?? Основные компоненты

- **Bot Engine:** Основной движок для обработки запросов и команд пользователей.
- **Database:** Хранение информации о картинах, пользователях и транзакциях.
- **Payment Gateway Integration:** Интеграция с платежными системами для обработки покупок.
- **Notification System:** Система уведомлений для информирования пользователей о новых картинах и предложениях.
